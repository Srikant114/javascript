Variables in JavaScript

A Javascript variable is just a name of storage location

We can Create variable in three ways

  • using let keyword.
  • using const keyword.
  • using var keyword.

Rules for naming variables

A variable name can contain any of the following Characters but must not start with a digit

  • Capital letters: A to Z
  • Small letters: a to z
  • Digits: 0 to 9 ( but we can't start variable name with digits )
  • Only two special characters that are: Dollor "$" and Underscore "_"
